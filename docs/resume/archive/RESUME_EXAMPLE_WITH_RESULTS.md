# 量化交易智能Agent系统 - 简历项目描述（含实际运行结果）

## 📊 项目概述

设计并实现了一个基于强化学习的智能交易Agent系统，采用经典的Agent循环架构（感知-决策-执行-评估），实现了从市场数据感知、智能决策、策略执行到性能评估的全流程自动化。系统支持多市场数据源（美股、A股、港股、加密货币），集成了市场状态识别、动态策略选择、智能仓位管理等核心功能。

---

## 🎯 核心技术亮点

### 1. 智能Agent架构设计 ⭐⭐⭐
- **感知-决策-执行-评估循环**：实现了完整的Agent生命周期管理
  - **感知模块**：自动获取多市场金融数据，支持数据缓存和异常检测
  - **决策模块**：基于市场状态和策略库进行智能决策，支持动态策略选择
  - **执行模块**：模拟真实交易环境，考虑手续费、滑点等交易成本
  - **评估模块**：多维度性能评估（收益率、夏普比率、最大回撤等）
- **配置驱动**：使用Pydantic实现类型安全的配置管理，支持参数验证和自动补全

### 2. 市场状态识别与动态策略选择 ⭐⭐
- **市场状态识别**：
  - 基于ATR、ADX、布林带等指标的市场状态分类器
  - 自动识别趋势市、震荡市、高/低波动等市场状态
  - **实际表现**：成功识别震荡市，准确推荐均值回归策略
- **动态策略选择**：
  - 根据市场状态自动选择最优策略（均值回归/动量策略）
  - **实际表现**：推荐策略在2020-2024年实现年化收益率8.46%

### 3. 自动参数优化 ⭐⭐
- **网格搜索**：实现了网格搜索自动参数优化
- **效果验证**：
  - 搜索了9个参数组合（窗口大小 × 阈值）
  - 将均值回归策略的年化收益率从**4.84%提升至16.45%**（提升3.4倍）
  - 夏普比率从0.26提升至**0.91**
  - 最大回撤控制在16.13%以内

### 4. 智能仓位管理 ⭐⭐
- **凯利公式**：基于历史胜率和盈亏比计算最优仓位
- **风险平价**：根据标的波动率动态调整仓位
- **实际表现**：
  - 交易胜率达到**69.57%**
  - 平均盈利2.28%，平均亏损2.74%

### 5. 强化学习支持 ⭐⭐⭐
- **Gymnasium环境接口**：实现了符合OpenAI Gym标准的交易环境
- **状态空间**：30+维特征向量（价格、技术指标、市场状态、仓位信息等）
- **多算法支持**：集成Stable-Baselines3，支持PPO、A2C、DQN等主流RL算法

## 🎯 面试重点（基于实际运行结果）

### Q1: 参数优化是如何提升策略表现的？
**A**: 通过网格搜索算法，我测试了9个参数组合（窗口大小15/20/25 × 阈值0.8/1.0/1.2），找到了最优组合（窗口15、阈值1.2）。这个组合将均值回归策略的年化收益率从4.84%提升至16.45%（提升3.4倍），夏普比率从0.26提升至0.91，最大回撤控制在16.13%以内。这说明参数优化对策略表现有显著影响。

### Q2: 市场状态识别如何工作？
**A**: 我实现了基于ATR、ADX、布林带等指标的市场状态分类器。在实际测试中，系统准确识别出2020-2024年的AAPL市场为震荡市，并推荐使用均值回归策略。推荐策略实现了8.46%的年化收益率，验证了识别系统的有效性。

### Q3: 如何评估策略的风险？
**A**: 我使用了多个指标：1) 最大回撤（控制在16.13%以内）；2) 夏普比率（0.91）；3) 年化波动率。另外，我还实现了智能仓位管理，使用凯利公式和风险平价方法，确保仓位大小与风险水平匹配。实际交易中胜率达到69.57%，说明风险控制有效。
