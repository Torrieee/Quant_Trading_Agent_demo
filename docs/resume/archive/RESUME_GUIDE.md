## 面试可能问到的问题

### 技术问题

1. **为什么选择 Agent 架构？**
   - Agent 架构能够清晰地组织量化交易的各个步骤
   - 便于扩展和维护
   - 体现了对系统设计的理解

2. **如何保证回测的准确性？**
   - 考虑了手续费、滑点等实际交易成本
   - 使用真实的历史数据
   - 实现了完整的交易逻辑

3. **参数优化的原理是什么？**
   - 网格搜索遍历参数空间
   - 根据评价指标（如夏普比率）选择最优参数
   - 体现了 Agent 的智能决策能力

### 业务问题

1. **策略的收益来源是什么？**
   - 均值回归策略：利用价格偏离均值的回归特性
   - 动量策略：利用趋势延续的特性

2. **如何评估策略的好坏？**
   - 综合考虑收益率、夏普比率、最大回撤等指标
   - 不仅要看收益，还要看风险

3. **项目的局限性是什么？**
   - 目前只支持日线数据
   - 回测没有考虑市场冲击成本
   - 策略相对简单，可以进一步优化
