## 📋 内容概览

### 项目定位
基于大模型的量化交易Agent系统 - 重点突出大模型在金融决策中的应用研究

### 核心亮点
1. **Function Calling框架** - LLM自动调用交易工具函数
2. **RAG系统** - 历史经验检索增强决策能力
3. **LLM策略生成** - 自然语言到策略的自动转换
4. **多模态Agent决策** - 结构化数据与LLM能力融合

### 项目成果
- 回测周期：2018-2025年（7年数据）
- 优化后年化收益率：16.45%，夏普比率：0.91
- 交易胜率：72.31%
- Function Calling成功验证LLM自动决策能力
